# 搭建vscode插件

   当木匠面临一再地重复制作同一样东西的任务时，他们会取巧。他们给自己建造夹具或模版。一旦他们做好了夹具，他们就可以反复制作某样工件。夹具带走了复杂性，降低了出错的机会，从而让工匠能够自由地专注于质量问题。Write Code That Writes Code ——《程序员修炼之道》

   所以每次新建页面或者组件时，都手动添加index.vue,index.stylus真是麻烦，而且vue中要init vue，引入index.stylus.那我们在vscode右键菜单中添加一个快捷创建文件夹文件的功能。

  代码已经有了(https://github.com/ARShl/autodirvueindex)，插件已经发布了(https://marketplace.visualstudio.com/items?itemName=alars.autodirvueindex)，可以参考下。以后创建再也不用手动做了。

  虽然开发的时间久了一些，不过还是挺好用的。
