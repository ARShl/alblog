# 写给大家看的设计书 3

  设计的某些方面需要在整个作品中重复。重复元素可能是一种粗字体，一条粗线，某个项目符号，颜色，设计要素，某种格式，空间关系等。你在工作中肯定已经用过重复。例如，所有标题都设置为相同的大小和粗细，在每一页底部添加一条半英寸的线，项目中每个列表都是用相同的项目符号，这些都是重复的例子。可以把重复认为一致性。在看一篇共有16页的新闻简报时，正是由于某些元素的重复，才使得这16页看上去属于同一个新闻简报。不过重复还不只是自然的一致，这是一种统一设计各个部分的有意识的行为。

  在生活中我们经常为了清晰和统一而需要重要性元素。上面图中的某些人是同一个队的，但是我们看不出来。他们衣服的重要性让我们马上就能看出来这些人属于某个组织。

  这个例子中，增加了一个重复的粗字体。看到电话号码时，下一步哪里，回到最前面，另一处粗文本嘛？这是设计人员常用来控制读者视线的一种视觉技巧，以使读者的注意力尽可能长地保持在页面上。

  原先可能使用过一些提供一致性的元素，可以把已有的重复元素取出，让这些元素效果更强，更主动。通过把重复表现的更明显。这样不仅页面看上去更有趣，还能在视觉上增强其条理性和一致性。

  重复有助于组织信息，可以帮助读者浏览各个页面，也有利于将设计中单独的部分统一起来。即使在一个只有一页的文档中，重复元素也可以建立一种连续性，将文档整个捆在一起。如果创建一个多页的，作为一个精美包装一部分的文档，充分使用重复就很关键。

  如果哪个元素引起了你的兴趣，就可以拿来用作重复元素！也许这是一个剪贴画，或者图片字体。为了实现重复，完全可以为此增加一些全新的东西。或者，也可以选择一个简单的元素，然后以多种不同的方式加以使用，比如说指定不同的大小，颜色或角度。有时候重复的项并不一定完全相同，而只是存在明确关联的紧密相关的对象。这张卡片使用的是居中。这么做让它避免看起来业余。

  重复性元素并不必须是图形或剪贴画。可以是空白，线，字体，对齐或者任何你有意重复的东西。

  有时，尽管只有重复元素这一点暗示，但这可能与整个元素有同样的效果。对于一个大家都熟悉的元素，可以尝试只包含其中的一部分，或者采用不同的方式加以利用。

  如果用户很熟悉其他营销素材中的某个图像，只取其一部分将有助于读者建立联系，重复还会为你的作品带来一种专业性和权威性，不管它看起来多有趣。它会使读者感觉有人在负责，因为重复显然是一种经过深思熟虑的设计决策。

  在上面的卡片中，标题的字体相同，但颜色不同。说明文字的字体各不相同，但都时尚。要确保有足够多重复的元素，这样差异才明显，而不是一团糟。

  最初，可以只是找出现有的重复，然后将其加强。熟悉了这种思想之后，再开始创建新的重复元素，来增强设计效果并提高信息的清晰度。重复就像是强调你的衣着。如果你穿着一件可爱的黑色晚礼服，戴着一顶别致的帽子，那你可能用红色高跟鞋，红色口红和一朵小花来突出晚礼服。

  要避免太多地重复一个元素，重复太多会让人讨厌。要注意对比的价值。例如，你想穿黑色晚礼服，配红帽子，红耳环，红口红，红手提包，红鞋和红上衣，重复就太滥了，现在的对比不再有统一效果，感觉上也让人不太舒服，太多的重复将混淆重点。
